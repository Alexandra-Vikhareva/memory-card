# Memory Card — игра на запоминание

Игра тренирует память и реакцию, а также служит полигоном для отработки `useState` и `useEffect` в React.

## Правила игры

- Перед игроком лежит набор уникальных карт (каждая карта встречается один раз).
- Задача — **нажать на каждую карту ровно один раз**.
- **После каждого клика** все карты перемешиваются.
- Нельзя нажимать на карту, которая уже была открыта ранее, — такая попытка не засчитывается.
- Игра заканчивается, когда все карты нажаты.
- Ведётся счёт:  
  - **Текущий счёт** = количество правильно нажатых карт.  
  - **Максимальный счёт (рекорд)** сохраняется между играми в `localStorage`.

## Технологии

- React (функциональные компоненты)
- Хуки: `useState`, `useEffect`
- Сборка: Vite
- Хранение рекорда: `localStorage`
