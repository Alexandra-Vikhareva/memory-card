# 🏴‍☠️ One Piece Memory Card Game

Игра на запоминание в стиле One Piece. Собирай дьявольские фрукты, тренируй память и ставь рекорды!

## 🌊 Описание игры

Игроку предстоит нажимать на карточки с изображениями дьявольских фруктов из вселенной One Piece. **Правила простые:**
- Каждую карточку можно нажать **только один раз**
- После каждого нажатия карточки **перемешиваются**
- Нельзя нажимать на уже открытую карточку — это поражение
- Нужно открыть **все карточки** подряд, чтобы победить
- Чем больше карточек открыто подряд — тем выше счёт

## 🎮 Игровой процесс

### 📋 Основные механики
- **Счёт и рекорд** — текущий счёт обновляется после каждого успешного клика, рекорд сохраняется в `localStorage`
- **Сложность** — выбор количества карт: 4, 8 или 12
- **Кнопка «Новая игра»** — сброс текущей игры
- **Загрузка из API** — карточки с фруктами загружаются с публичного API One Piece

## 🎯 API
Проект использует публичное API:
- `GET /v2/fruits/en/count` — получение количества фруктов
- `GET /v2/fruits/en/{id}` — получение данных и картинки фрукта

Загружаются только фрукты с валидными изображениями.
